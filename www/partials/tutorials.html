Step 1
  • This page describes the various elements and controls on the page.
  • This tutorial will give you a basic walk through on creating a page / dashboard.
  • Start by clicking on the top left corner as seen in image labeled step 1. This will bring up the page list.
Step 2
  • On the left hand side you see a list of existing pages.
  • Clicking on any of these will load the page.
  • Towards the top there is a '+' icon as shown in the image. Click on it to start building a new dasboard.
Step 3
  • When creating a new page you are brought into edit mode and a new side panel appears.
  • Explore to see more controls to manipulate the page.
  • You can add additional graphs to a pod by clicked the '+' icon seen to the right.
  • Search for a metric to add to the graph in a pod.
Step 4
  • After you've found a metric you can drag onto one of the graph place holders in a pod.
  • You may drag multiple metrics on to the same graph.
  • You can then change the graph type, graph size in terms of hieght.
Step 4.1 (optional)
  • Once a metric has been added to the graph you can change options for that metric in the panel below the graph. There is one panel per metric added.
  • Options here pertain to each metric and the data itself.
  • All TSDB query options are available here along with other options as seen in the image.
Step 4.2 (optional)
  • More pods can be added by double-clicking the appropriate region as shown in the image.
  • Additionally pods, columns, rows, and graphs can be moved to desired postions.
  • Graphs can be re-ordered within a pod.
  • Pods can be re-ordered within a column and moved to other columns.
  • Columns can be re-ordered within a row and moved between rows.
  • Rows can be re-ordered within the page.
Step 5 and Step 6
  • After the desired dashboard has been constructed, enter a name and ID for the page.
  • The ID cannot contain any special characters dashes and underscores are allowed.
  • The ID is used in the URL when referencing the page.
  • Finally click the save button to save the page.
